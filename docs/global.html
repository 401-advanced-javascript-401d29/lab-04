writeBuffer(array)

This function splits the array into individual characters and write them to the buffer Important because each character has to be written to the buffer individually. Cannot send entire array at the same time. buffer.write(string, offset) takes in the string and then the offset. If offset is not incremented each newly added character will overwrite the previous character. buffer.write also can take in length and encoding. If nothing is specified these default to buffer.length-offset and utf8 for encoding. This is why you don't have to use charCodeAt().
